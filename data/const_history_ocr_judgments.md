# Constitution history — OCR / interpretive judgment log

Every place where we resolved an OCR ambiguity or made an interpretive call while
reconstructing the historical text. Recorded so each is auditable and can be
**confirmed by a later clean publication** without relying on judgment.

Method:
1. **Backstop first** — if the language survives unchanged into a clean publication
   (present-day constitution; 1961/1973/1981 Blue Books), use that clean text. No
   judgment needed; record the confirming source.
2. **Reconcile + log** — for portions that changed, reconcile the OCR witnesses
   (session law + integrated Blue Book) and record each fix below.

Columns: provision · effective date · location (quoted) · OCR reading · applied
reading · basis (backstop source, or "judgment").
