ndlaw

North Dakota primary law as an MCP server — free, public, no account required.

This endpoint gives AI assistants (Claude, and any MCP-capable client) citation-verified access to North Dakota primary legal authority:

  • Supreme Court & Court of Appeals opinions, 1889–present (~19,800 opinions, full text, citation graph)
  • Constitution of North Dakota — point-in-time versions, amendment history, 1889-numbering crosswalk
  • N.D. Century Code, court rules, and administrative code — versioned provisions
  • Attorney General opinions (1942–present) and Judicial Ethics Advisory Committee opinions

Connect

MCP endpoint (Streamable HTTP):

https://ndlaw.org/mcp

Claude Code:

claude mcp add --transport http ndlaw https://ndlaw.org/mcp

Claude.ai / Claude Desktop: add a custom connector with the URL above. Any other MCP client: point its HTTP transport at the same URL.

Fair use. This is a shared public server on modest hardware. Per-IP limits apply (roughly one request per second sustained); heavier traffic is temporarily banned, not queued. If you need bulk data, don't scrape the API — download the whole database below.

Bulk data

The complete SQLite databases are free to download and republish (CC0): github.com/jet52/ndlaw/releases. Run the server locally with the same package.

Caveats

The data is compiled from public sources (court PDFs, CourtListener, ndcourts.gov, ndlegis.gov) with extensive validation, but it is not an official publication of the State of North Dakota or its courts. Verify against the official source before filing.

Related: ndconst.org — the Constitution of North Dakota as an annotated wiki, built from the same corpus.
